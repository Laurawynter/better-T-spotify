Material Icons

2,100+ ready-to-use React Material Icons from the official website.

Break free from being a feature factory. Try Aha! Develop for free.ads via Carbon

The following npm package, @mui/icons-material, includes the 2,100+ official Material Icons converted to SvgIcon components.

The @mui/icons-material package depends on @mui/material, which requires Emotion packages. If you don't use Material UI in your project yet, install the icons package with: npm install @mui/icons-material @mui/material @emotion/styled @emotion/react.

See the Installation page for additional docs about how to make sure everything is set up correctly.
